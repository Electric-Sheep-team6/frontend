# フロントエンド基本設計

## 1. 画面一覧

| 画面名      | 概要                           |
| -------- | ---------------------------- |
| ログイン画面   | メールアドレスまたはgoogle,SNSアカウントでログインする。   |
| 新規登録画面   | アカウントを作成する。                  |
| ホーム画面    | タイムラインやAIからのおすすめ、思い出通知を表示する。 |
| 投稿作成画面   | 写真・動画・音声・テキスト・感情を記録する。       |
| 投稿詳細画面   | 投稿内容やコメント、共同編集者を確認する。        |
| 検索画面     | AI検索・タグ検索・人物検索・感情検索を行う。      |
| タイムライン画面 | 過去の投稿を時系列で閲覧する。              |
| 感情グラフ画面  | 日々の感情をグラフで可視化し、特定の日へ移動する。    |
| ルーム画面    | 家族や友人との共有ルームを管理する。           |
| プロフィール画面 | ユーザー情報や投稿数などを確認・編集する。        |
| 通知画面     | 「○年前の今日」やルーム更新通知などを表示する。     |
| 設定画面     | アカウント設定・通知設定・プライバシー設定を行う。    |

# 6. 入力項目・バリデーション

| 項目      | 必須 | バリデーション  |
| ------- | -- | -------- |
| メールアドレス | ○  | メール形式    |
| パスワード   | ○  | 8文字以上    |
| 投稿本文    | ×  | 最大1000文字 |
| 感情      | ○  | 1つ選択     |
| タグ      | ×  | 最大10件    |
| 写真      | ×  | JPEG・PNG |
| 動画      | ×  | MP4      |
| 音声      | ×  | 最大5分     |

# 9. エラー表示・ローディング表示

## ローディング

* ログイン中
* 投稿送信中
* AI検索中
* データ取得中

画面中央にスピナーを表示する。

---

## エラー表示

| エラー       | 表示内容                       |
| --------- | -------------------------- |
| ログイン失敗    | メールアドレスまたはパスワードが正しくありません。  |
| 通信エラー     | 通信に失敗しました。時間をおいて再度お試しください。 |
| 投稿失敗      | 投稿できませんでした。                |
| AI検索失敗    | AI検索に失敗しました。               |
| ファイルサイズ超過 | ファイルサイズが上限を超えています。         |

エラーはSnackbarまたはダイアログで表示し、必要に応じて再試行ボタンを表示する。

# 11. API連携

バックエンドとの通信仕様を整理する。
( 仮で書いているのでバックエンド班に後ほど変更してもらう)

| Method | Endpoint       | 内容      |
| ------ | -------------- | ------- |
| POST   | /auth/login    | ログイン    |
| POST   | /auth/register | 新規登録    |
| GET    | /posts         | 投稿一覧取得  |
| POST   | /posts         | 投稿作成    |
| GET    | /posts/{id}    | 投稿詳細取得  |
| PUT    | /posts/{id}    | 投稿更新    |
| DELETE | /posts/{id}    | 投稿削除    |
| GET    | /search        | AI検索    |
| GET    | /emotion       | 感情グラフ取得 |
| GET    | /notifications | 通知取得    |
